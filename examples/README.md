# examples

このディレクトリには、最低限のサンプルを入れています。

- `sample-media/`
  - 記事メディア向けの例
  - 長文、mixed-script 見出し、URL、caption を確認しやすい
- `sample-saas/`
  - SaaS / 管理画面向けの例
  - フォーム、表、ラベル、mixed navigation を確認しやすい

`preview.html` はそのままブラウザで開けます。  
相対パスのCSSを安定して読み込みたい場合は、リポジトリのルートで簡易サーバーを立ててください。

```bash
python -m http.server 8000
```
